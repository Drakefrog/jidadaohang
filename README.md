吉大导航
===========

吉大导航，一款可以在吉大内部使用的导航软件。来自2013年大创项目。

##1.我是谁？

这是一个吉大的大创项目，合作人包括张鸿飞、韩增易、魏志杰、谭振兴以及王多民（其实你什么都没做是吧），以及特别要感谢的嘉宾**程紫东**师哥。由衷感谢指导老师**孙永雄**副教授。
这个项目的名称按下不表，这个APP的名称叫做：“吉大导航”。

##2.我是干什么的？

项目主要实现了以下的功能：

+ APP使用者的位置确定
+ 吉大校园内部任意楼宇与当前位置的导航路线
+ 吉大校园内楼宇、教学或办公地点的名称模糊搜索

以上
